Theory: Data Cleaning & Visualisation

The skills that turn raw data into business intelligence

🧹 The 5 Most Common Dirty Data Problems

  1. Numbers stored as text — SUM returns 0.
    Fix: Data → VALUE() function.
  2. Inconsistent date formats — sorting fails.
    Fix: DATEVALUE(), or reformat all as DD/MM/YYYY.
  3. Leading/trailing spaces — lookups fail silently.
    Fix: TRIM(A2)
  4. Mixed case text — "COFFEE" ≠ "coffee".
    Fix: =PROPER(A2) or =UPPER(A2)
  5. Blank rows & merged cells — pivot tables break.
    Fix: Find & Select → Go To Special → Blanks.

📊 Chart Types — Pick the Right One

  • Column / Bar Chart — compare categories (sales by product)
  • Line Chart — show trends over time (monthly revenue)
  • Pie / Doughnut — show parts of a whole (% of revenue by service) — use sparingly
  • Scatter Plot — show correlation (hours worked vs. revenue)
  • Combo Chart — two metrics on one chart (revenue bars + margin % line)

🔧 Essential Cleaning Functions

# Remove extra spaces from text in cell A2
=TRIM(A2)

# Convert text "1,234" to number 1234
=VALUE(SUBSTITUTE(A2,",",""))

# Standardise text to Title Case
=PROPER(A2)

# Convert text date "Jan 15 2024" to Excel date
=DATEVALUE(A2) ← then format as date

# Check if cell is empty
=ISBLANK(A2) ← returns TRUE if blank

Quick Reference: Which Chart for Which Question?
Question Use This Chart Not This
How did X change over time? Line / Area Pie
What share does each part hold? Pie / Donut Line
Compare categories side by side Bar (clustered) Pie
Rank items from most to least Bar (sorted) Scatter

Takeaway: Before inserting a chart in Excel, ask yourself: "What question am I answering?" The answer tells you the chart type. Picking the wrong type doesn't just look bad — it tells a false story.

📥 ADVANCED: Import Data with Data → Get Data → From Text/CSV How To

Before you can clean data, you need to get it into Excel. Raw exports often arrive as .csv or .txt files. Excel's built-in import wizard gives you full control over how the file is read — so dates, numbers, and text all land in the right columns from the start.

📋 Step-by-Step: Import a CSV

  1. Open Excel → Data tab → Get Data → From File → From Text/CSV
  2. Browse to your file and click Import. A preview window opens.
  3. Check the File Origin (encoding) — use 65001: Unicode (UTF-8) for files with umlauts or accents (e.g., ü, é).
  4. Check the Delimiter — usually Comma for .csv or Tab or Whitespace for .txt exports.
  5. Click Transform Data to open Power Query Editor — or click Load to import directly.
  6. In Power Query: right-click any column header to Change Type (e.g., force a column to Date or Whole Number).
  7. Click Close & Load — data lands in a new sheet as a structured Table.

⚠️ Common Import Pitfalls

  • Wrong encoding — if you see Ã¼ instead of ü, change File Origin to UTF-8 in step 3.
  • Dates imported as text — if the date column stays as text after import, use Power Query → Change Type → Date.
  • Numbers with commas as decimals (e.g., 14,99) — Excel may read them as text. Fix with VALUE(SUBSTITUTE(A2,",",".")) after import.
  • Extra header rows — if the file has a banner row (like "Export: 01-07-2025") before the real headers, remove it in Power Query: Home → Remove Rows → Remove Top Rows.
  • All columns merged into one — wrong delimiter selected. Go back and try Comma, Semicolon, or Tab.
🆚 Direct Open vs. Import Wizard — When to Use Which
Situation Approach
Quick look at a clean, simple CSV Double-click to open directly
File has special characters (ü, ñ, ç) Data → From Text/CSV (set UTF-8)
Dates or numbers landing as text Data → From Text/CSV → Transform Data
File has extra header/banner rows Data → From Text/CSV → Remove Top Rows
Source file updates regularly Import → creates a refreshable connection

Takeaway: Double-clicking a CSV is fast — but it lets Excel guess the format, which causes silent errors with dates and special characters. Use Data → Get Data → From Text/CSV for any file you plan to clean or analyse. Hands-On: Clean + Visualise

~90 minutes total · Part 1: Clean (45 min) · Part 2: Visualise (45 min)

📁
Your File: [company_name]_visualization.xlsx Open your company's operations dataset (e.g. bytebite_visualization.xlsx). This file contains real-looking but fictional operational data — orders, bookings, surveys, or project records depending on your company. It is intentionally messy: your job today is to clean it and then visualise it.

Part 1: Clean Your Dataset

  1. Open [company_name]_visualization.xlsx. Make a copy of the raw data sheet (right-click tab → Move or Copy) — never clean the original!
  2. Rename the copy to "Clean Data".
  3. Format the data as Table -- Choose one of the predefined Light Table Styles.
  4. Insert a new column called "Notes" to document any issues you find.
  5. Scan each column: identify all inconsistencies. Write them down in a notes cell.
  6. Fix number-as-text columns: Use VALUE() or Text to Columns method.
  7. Add two columns "First Name" and "Last Name" and use "Text-to-Columns" to split full names. Make sure to set the destination cell correctly.
  8. Fix date inconsistencies: Standardise to one format using DATEVALUE() or manual Find & Replace.
  9. Fix text inconsistencies: Apply TRIM() and PROPER() where needed.
  10. Add a column called "Data Quality Check": use ISBLANK() or IF() to flag any remaining issues.
  11. Use Conditional Formatting to highlight any rows with issues in red (e.g., if "Data Quality Check" = "Issue").

Part 2: Build Your Charts

  1. Select your clean data. Go to Insert → Recommended Charts — see what Excel suggests.
  2. Build a Column Chart showing your primary metric over time (e.g., revenue by month, units by product).
  3. Click the chart → Chart Design → Change Colors → apply your brand colours.
  4. Add proper axis labels and a chart title that tells a story (e.g., "Revenue Growth — Q1 2024").
  5. Build a second chart — a Line Chart or Combo Chart showing a different insight (e.g., margin trend).
  6. Create a new sheet called "Dashboard". Move/copy both charts here. Arrange them neatly.
  7. Add a text box with a 1-sentence key insight for each chart. What is the data telling you?
