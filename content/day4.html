Theory: Data Cleaning & Visualisation

The skills that turn raw data into business intelligence

🧹 The 5 Most Common Dirty Data Problems

  1. Numbers stored as text — SUM returns 0. Fix: Data → Text to Columns, or VALUE() function.
  2. Inconsistent date formats — sorting fails. Fix: DATEVALUE(), or reformat all as DD/MM/YYYY.
  3. Leading/trailing spaces — lookups fail silently. Fix: =TRIM(A2)
  4. Mixed case text — "COFFEE" ≠ "coffee". Fix: =PROPER(A2) or =UPPER(A2)
  5. Blank rows & merged cells — pivot tables break. Fix: Find & Select → Go To Special → Blanks.

📊 Chart Types — Pick the Right One

  • Column / Bar Chart — compare categories (sales by product)
  • Line Chart — show trends over time (monthly revenue)
  • Pie / Doughnut — show parts of a whole (% of revenue by service) — use sparingly
  • Scatter Plot — show correlation (hours worked vs. revenue)
  • Combo Chart — two metrics on one chart (revenue bars + margin % line)

🔧 Essential Cleaning Functions

# Remove extra spaces from text in cell A2
=TRIM(A2)

# Convert text "1,234" to number 1234
=VALUE(SUBSTITUTE(A2,",",""))

# Standardise text to Title Case
=PROPER(A2)

# Convert text date "Jan 15 2024" to Excel date
=DATEVALUE(A2) ← then format as date

# Check if cell is empty
=ISBLANK(A2) ← returns TRUE if blank

Find & Replace

Ctrl+H — bulk-fix inconsistent entries (e.g., "EUR" → "€")

Text to Columns

Data → Text to Columns — split "First Last" into two cells, or force numeric conversion

Freeze Panes

View → Freeze Panes → Freeze Top Row — keeps headers visible when scrolling

Conditional Formatting

Home → Conditional Formatting — highlight cells above/below threshold automatically

Quick Reference: Which Chart for Which Question?
Question Use This Chart Not This
How did X change over time? Line / Area Pie
What share does each part hold? Pie / Donut Line
Compare categories side by side Bar (clustered) Pie
Rank items from most to least Bar (sorted) Scatter

Takeaway: Before inserting a chart in Excel, ask yourself: "What question am I answering?" The answer tells you the chart type. Picking the wrong type doesn't just look bad — it tells a false story.

Hands-On: Clean + Visualise

~90 minutes total · Part 1: Clean (45 min) · Part 2: Visualise (45 min)

📁
Your File: [company_slug]_visualization.xlsx Open your company's operations dataset (e.g. bytebite_visualization.xlsx). This file contains real-looking but fictional operational data — orders, bookings, surveys, or project records depending on your company. It is intentionally messy: your job today is to clean it and then visualise it.

Part 1: Clean Your Dataset

  1. Open [company_slug]_visualization.xlsx. Make a copy of the raw data sheet (right-click tab → Move or Copy) — never clean the original!
  2. Scan each column: identify all inconsistencies. Write them down in a notes cell.
  3. Fix blank rows: Home → Find & Select → Go to Special → Blanks → Delete rows.
  4. Fix number-as-text columns: Use VALUE() or Text to Columns method.
  5. Fix date inconsistencies: Standardise to one format using DATEVALUE() or manual Find & Replace.
  6. Fix text inconsistencies: Apply TRIM() and PROPER() where needed.
  7. Add a column called "Data Quality Check": use ISBLANK() or IF() to flag any remaining issues.

Part 2: Build Your Charts

  1. Select your clean data. Go to Insert → Recommended Charts — see what Excel suggests.
  2. Build a Column Chart showing your primary metric over time (e.g., revenue by month, units by product).
  3. Click the chart → Chart Design → Change Colors → apply your brand colours.
  4. Add proper axis labels and a chart title that tells a story (e.g., "Revenue Growth — Q1 2024").
  5. Build a second chart — a Line Chart or Combo Chart showing a different insight (e.g., margin trend).
  6. Create a new sheet called "Dashboard". Move/copy both charts here. Arrange them neatly.
  7. Add a text box with a 1-sentence key insight for each chart. What is the data telling you?
✅
Quality Check Change one data value in your cleaned dataset. Both charts should update automatically. If they do — your charts are linked to the live data. If not — you may have pasted values instead of using cell references.
